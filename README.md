# Frangit Systems

Landing page for Frangit Systems - IT & Digital Services.

## Project Purpose

This project is the first digital foundation for the Frangit Systems brand. Its current goal is to present the business professionally, communicate services clearly, and generate contact through WhatsApp, Instagram, and email.

## Current Stack

- React
- Vite
- CSS

## Useful Commands

Use full paths on this Windows setup when PowerShell does not resolve `node` or `npm` correctly.

### Start development server

```powershell
& "C:\Program Files\nodejs\npm.cmd" run dev
```

### Create production build

```powershell
& "C:\Program Files\nodejs\npm.cmd" run build
```

### Run lint

```powershell
& "C:\Program Files\nodejs\npm.cmd" run lint
```

## Project Structure

- `src/App.jsx`: landing page structure and content
- `src/App.css`: section-level styling
- `src/index.css`: global variables and base styles
- `public/brand/`: brand assets used by the site
- `PROJECT_CONTEXT.md`: business and branding context
- `DEPLOYMENT_GUIDE.md`: deployment notes for publishing later
